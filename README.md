# iCE40-Addon-Boards
A collection of add-on boards containing an iCE40 FPGA and the required supporting logic. Used to add FPGA functionality to existing hardware platforms, particularly microcontrollers. The boards are currently a work in progress. Currently, development prototypes are being tested, and the add-on boards will be designed once testing is complete.

## Planned platforms for add-on boards
-Arduino Uno
-Raspberry Pi Pico
-Seeed XIAO
-Adafruit Feather?
-QWIIC-only breakout?
